# SmartShop - Point of Sale System

A comprehensive Point of Sale (POS) system built with PHP and MySQL for retail businesses.

## Features

- **User Management**: Multi-role authentication (Admin, Manager, Cashier)
- **Product Management**: Inventory tracking, categories, pricing
- **Sales Processing**: Complete POS interface with receipt generation
- **Returns & Exchanges**: Process returns with reason tracking
- **Reporting**: Sales reports, shift summaries, performance analytics
- **Multi-language Support**: English and Kinyarwanda
- **Staff Scheduling**: Shift management and tracking

## Installation

1. Clone the repository:
```bash
git clone https://github.com/ukvalens/SmartShop.git
```

2. Set up your web server (Apache/Nginx) to point to the project directory

3. Create a MySQL database and import the schema

4. Configure database connection in `config/database.php`

5. Set up proper permissions for uploads directory

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- GD extension for image processing

## Usage

1. Access the application through your web browser
2. Login with admin credentials
3. Set up products, categories, and users
4. Start processing sales
